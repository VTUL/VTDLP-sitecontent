About This Software

The Virginia Tech Digital Libraries Platform (VTDLP) is a cloud-native, digital libraries solution supporting preservation and access while taking advantage of the efficiencies of Amazon Web Services. It is a collection of CloudFormation templates, code implementing microservices to support digital library functions, and web applications providing user access. It is a free and Open Source collection of cloud configurations and microservices supporting a wide range of digital preservation, access, and digital collection management, activities.

For more information:

  • See our demo site or an existing digital library
  • Visit our Github
  • View our documentation
  • Partner with us by contributing
