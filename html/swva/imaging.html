IMAGING STANDARDS

The following tables aggregate standards based for digital preservation based on consortial requirements and based on the work of national governance bodies and interest groups.

  • Printed and handwritten text
  • Printed photographs
  • Large format maps and posters
  • Negatives and transparencies
  • Books
  • Newspapers
  • Microfilm
  • Microfiche

 

Printed and handwritten text
Scan Type Bit Depth Resolution (ppi/dpi) Color space Scale File format
Black and white print 1-bit 600 Bitonal 1:1 TIFF with Group 4 lossless compression AND PDF/A
Black and white  + illustrations, or handwritten 8-bit 400 Grayscale 1:1 Uncompressed TIFF AND PDF/A
Text with color images, or handwritten 24-bit 400 RGB 1:1 Uncompressed TIFF AND PDF/A

 

Printed photographs
Type Bit Depth Resolution (ppi/dpi) Color space Scale File format
Black and white 8-bit 600 Grayscale 1:1 Uncompressed Tiff
Color 24-bit 600 RGB 1:1 Uncompressed TIFF

 

Large format maps and posters
Type Bit Depth Resolution (ppi/dpi) Color space Scale File format
Black and white 1-bit 5000 - 10,000 pixels on the longest side Bitonal 1:1 Uncompressed TIFF
Grayscale 8-bit 5000 - 10,000 pixels on the longest side Grayscale 1:1 Uncompressed TIFF
Color 24-bit 5000 - 10,000 pixels on the longest side RGB 1:1 Uncompressed TIFF

 

Negatives and transparencies
Type Bit Depth Resolution (ppi/dpi) Color space Scale File format
Black and white 8-bit 5000 pixels on the longest side* Grayscale 1:1 Uncompressed TIFF
Color 24-bit 5000 pixels on the longest side* RGB 1:1 Uncompressed TIFF
 

*Common film sizes and corresponding resolutions to achieve 5000 pixels on the longest side.
